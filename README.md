# Yashaswini Badarla — Portfolio

A responsive, dependency-free personal portfolio for AI, data, and software engineering roles.

## Live project links

- Espresso AI: https://espresso-ai.vercel.app
- GitHub: https://github.com/Yashaswini1233

## Run locally

Open `index.html` directly in a browser, or run:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deploy to Vercel

1. Create a new GitHub repository, for example `yashaswini-portfolio`.
2. Upload all files from this folder.
3. Import the repository into Vercel.
4. Choose **Other** as the framework preset.
5. Leave build command and output directory empty.
6. Deploy.

This site has no framework, package installation, environment variables, or build process.
